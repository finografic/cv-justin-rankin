# CV Justin Rankin <span style="color:#AAAAAA">(v2)</span>

### Initialize, installing dependencies:

```sh
yarn
```

### Run local development mode:

```sh
# watch code + eslint watch
yarn go

# watch code only
yarn start
```

### Create production build:

```sh
# build only
yarn build

# build + serve on localhost:5000
yarn serve
```
